# ai

AI resources repository

### AI Online tools & scope

| TOOL                                     | BEST FOR                                               | AVOID FOR                          |
| ---------------------------------------- | ------------------------------------------------------ | ---------------------------------- |
| [ChatGPT](https://chatgpt.com/)          | Creative writing, brainstorming, versatility           | Fact-checking, real-time info      |
| [Claude](https://claude.ai/new)          | Long documents, nuanced analysis, complex instructions | Quick, punchy responses            |
| [Perplexity](https://www.perplexity.ai/) | Research with citations, fact-checking, current info   | Creative writing, brainstorming    |
| [DeepSeek](https://chat.deepseek.com)    | Coding, math, technical problem-solving                | Creative content, emotional nuance |
| Copilot                                  | Microsoft 365 workflow, document/data work             | Work outside Microsoft ecosystem   |
| Gemini                                   | Google ecosystem, real-time info, multimodal           | Deep analytical reasoning          |
