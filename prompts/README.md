### Effective Prompts

| ELEMENT              | WHAT IT DOES                                           | EXAMPLE                                                                 |
| -------------------- | ------------------------------------------------------ | ----------------------------------------------------------------------- |
| **1. Role**          | Sets the perspective or expertise the AI should adopt  | "You are an experienced HR professional with 15 years in tech startups" |
| **2. Context**       | Provides background information for relevant responses | "I'm launching a mobile app for busy parents to meal plan"              |
| **3. Task**          | Specifies the concrete action you want completed       | "Create a 5-week onboarding plan for new support agents"                |
| **4. Output Format** | Defines how you want the information presented         | "Present as a table with three columns. Provide 5 rows."                |
| **5. Tone**          | Indicates the voice or style to use                    | "Conversational and encouraging, like a mentor"                         |
